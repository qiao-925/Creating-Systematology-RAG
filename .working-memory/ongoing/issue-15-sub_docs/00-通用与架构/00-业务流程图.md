**关键处理**：
- CLD 层：3个Agent（政策/经济/社会）并行提取 → JSON Schema输出
- CLD 层全自动：节点归并（阈值0.8自动处理）→ 冲突检测 → 高分歧送裁判Agent
- FCM 层：语言权重自动聚合（均值），输出权重矩阵 + 置信度标记
- D2D 层：敏感性分析自动识别杠杆点，按影响力+置信度排序
- 全程无人工介入，系统自动完成提取→融合→仿真→分析
