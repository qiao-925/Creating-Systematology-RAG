# 决策哲学层

## 核心信条

**决策的首要问题是分类，而非选择。**

---

## 第一性原理

### 原理1：因果可知性

所有决策情境的差异，归根结底是**"因果关系的可知程度"**。

```
已知因果  →  执行问题（怎么做）
未知因果  →  探索问题（做什么）
```

### 原理2：时间不对称性

决策质量与决策速度之间存在**结构性张力**。

```
快速决策  →  依赖模式/直觉  →  适用于已知情境
慢速决策  →  依赖分析/推理  →  适用于未知情境
```

**关键洞察**：
- 在已知情境强行慢思考 = 浪费资源
- 在未知情境强行快决策 = 系统性错误

### 原理3：涌现性

复杂系统中，**整体行为不可还原为个体行为之和**。

```
个体理性  ≠  集体理性
局部最优  ≠  全局最优
当前最优  ≠  长期最优
```

**推论**：
- 无法通过分析局部来预测整体
- 必须通过实验来"感知"涌现行为
- 计划必须短周期、可调整

## 设计哲学

### 哲学1：域敏感（Domain Sensitivity）

**不存在通用最佳实践，只有域适配实践。**

| 情境特征 | 哲学 |
|----------|------|
| 稳定可预测 | 标准化、自动化、零决策 |
| 分析可解 | 多方案、延迟承诺、专家验证 |
| 涌现演化 | 探测实验、放弃预测、快速响应 |
| 崩溃危机 | 先行动、分层响应、事后学习 |

### 哲学2：分层自治（Layered Sovereignty）

**决策权应分层持有，通过契约而非协调来对齐。**

```
战略层定目标    ← 不干预战术层"怎么做"
战术层定方案    ← 不干预反应层"怎么跑"
反应层定执行    ← 反馈状态但不请求批准
```

**关键洞察**：
- 跨层实时协调成本极高
- 契约边界清晰比完美对齐更重要

### 哲学3：重心聚焦（Focus Economics）

**资源分配遵循幂律，而非均匀分布。**

```
80%资源 → 20%关键决策（Type 3）
20%资源 → 80%常规决策（Type 1/2）
```

**推论**：
- 必须能识别什么是关键
- 必须敢于用默认值处理非关键
- 关键决策值得等待和深度研讨

### 哲学4：人机共生（Human-AI Symbiosis）

**人类与AI的优势边界清晰，不应互相僭越。**

| 人类擅长 | AI擅长 |
|----------|--------|
| 定义问题 | 执行计算 |
| 价值判断 | 生成选项 |
| 意义建构 | 模式识别 |
| 创造性突破 | 一致性保持 |

**关键洞察**：
- 让AI做选择 = 放弃人类责任
- 让人类做计算 = 浪费认知资源

## 反模式（Anti-Patterns）

### 反模式1：域误判

**症状**：
- 在Complex域做详细长期计划
- 在Clear域要求多方案对比
- 在Complicated域追求立即决策

**后果**：
- 计划失效，资源浪费
- 决策瘫痪，机会成本
- 草率决策，后期返工

### 反模式2：层级越界

**症状**：
- 战略层干预代码实现细节
- 反应层质疑战略方向
- 缺乏契约，依赖实时协调

**后果**：
- 决策冲突，执行混乱
- 责任不清，互相推诿
- 效率低下，响应迟缓

### 反模式3：重心分散

**症状**：
- 所有决策同等对待
- 非关键决策占用大量资源
- 关键决策草率通过

**后果**：
- 关键决策质量低
- 系统脆弱性高
- 整体效率低下

### 反模式4：人机错位

**症状**：
- 人类做重复性计算
- AI做价值判断
- 责任归属模糊

**后果**：
- 人类认知疲劳
- AI输出不可信
- 系统失控风险

## 元认知框架

### 决策前的决策

每次决策前，先回答：

```
1. 这是什么域？
   → 决定"用什么方式决策"

2. 这是什么层级？
   → 决定"谁来做决策"

3. 这是什么类型？
   → 决定"投入多少资源"

4. 谁和谁协作？
   → 决定"人机如何分工"
```

### 反馈闭环

```
决策 → 执行 → 观察 → 学习 → 更新认知 → 下一轮决策
   ↑___________________________________________↓
```

**关键**：
- Complex域：循环周期短（周级）
- Complicated域：循环周期中等（月级）
- Clear域：循环周期长（季度/年级），主要优化执行效率

---

## 适用范围与边界

### 适用

- 结构化决策场景（有明确决策点）
- 分层组织（战略/战术/执行分离）
- 人机协作系统（AI辅助决策）

### 不适用

- 完全非结构化的混乱
- 纯创意/艺术创作
- 极端高压无思考时间
