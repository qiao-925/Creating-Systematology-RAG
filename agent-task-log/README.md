## 命名规范

```

```

- `YYYY-MM-DD`: 日期
- `N`: 当日序号
- `task_type`: 任务类型（implementation/refactor/bugfix/optimization/analysis/documentation）
- `文档类型`: 完成总结/快速摘要/详细过程等
