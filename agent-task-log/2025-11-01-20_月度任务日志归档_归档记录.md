## 💡 归档规则建立

### 月度归档机制
1. **归档时机**：每月结束后，将上一月份的所有任务日志归档到对应月份目录
2. **归档目录命名**：`YYYY-MM/`（如 `2025-10/`、`2025-11/`）
3. **归档文件**：所有以 `YYYY-MM-` 开头的 `.md` 文件
4. **保留文件**：模板文件（README.md、TEMPLATE.md、人机协作范式.md）和当前月份的任务日志

### 归档流程
```bash
# 每月归档操作（示例：11月归档到12月时）
1. 创建目录：mkdir 2025-11
2. 移动文件：mv 2025-11-*.md 2025-11/
3. 验证结果：确认文件已正确归档
```
