## 2. 对话起点：Resume 的双重含义

用户在 Cursor 中看到 "resume"，询问其含义。

| 词性 | 含义 | 发音 |
|------|------|------|
| 动词 | 恢复、继续 | /rɪˈzuːm/ |
| 名词 | 简历（美式英语） | /ˈrezəmeɪ/ |

**Cursor 语境**：恢复执行被中断的任务。

**名词来源**：法语 résumé（摘要），这引出了英语词汇来源的话题。

## 3. 英语：一门"混血语言"

### 3.1 词汇来源占比

| 来源 | 占比约 | 典型词汇 |
|------|--------|----------|
| 拉丁语 | 29% | computer, virus, data |
| 法语 | 29% | restaurant, café, résumé |
| 日耳曼语（本族） | 26% | water, house, come, go |
| 希腊语 | 6% | philosophy, democracy |
| 其他 | 10% | 各种语言借词 |

### 3.2 历史原因

1. **日耳曼根基**：英语本是日耳曼语族
2. **1066 诺曼征服**：法国人统治英国 300 年，大量法语涌入
3. **文艺复兴**：学术界从拉丁/希腊语借词
4. **殖民扩张**：从全球吸收词汇

### 3.3 有趣现象：双胞胎词

同一概念常有不同来源的同义词：
- 日耳曼词（口语）：begin, buy, freedom
- 拉丁/法语词（正式）：commence, purchase, liberty

## 4. 技术命名趣闻

### 4.1 JavaScript：蹭热度的经典案例

**1995年时间线**：
1. Brendan Eich 用 **10 天** 设计出浏览器脚本语言
2. 最初叫 Mocha → 改名 LiveScript
3. Netscape 与 Sun 合作，为蹭 Java 热度改名 **JavaScript**
4. 技术上与 Java **毫无关系**

**官方吐槽**：
> "JavaScript 之于 Java，就像 carpet（地毯）之于 car（汽车）"

一个 10 天赶工 + 蹭热度命名的语言，成为世界上使用最广泛的编程语言之一。

### 4.2 Git：Linus 的自嘲

**背景**：
- 2005 年 BitKeeper 撤销 Linux 内核的免费授权
- Linus Torvalds **用 2 周** 写出 Git 初版

**命名解释**（Linus 原话）：
> "I'm an egotistical bastard, and I name all my projects after myself. First Linux, now Git."

**Git 的多重含义**：
| 解释 | 说明 |
|------|------|
| 英式俚语 | "讨厌鬼、混蛋" |
| 官方解释（能用时） | Global Information Tracker |
| 官方解释（不能用时） | Goddamn Idiotic Truckload of sh*t |

### 4.3 其他趣闻

- **Python**：Guido 是《Monty Python》喜剧粉丝，跟蛇没关系
- **Bluetooth**：维京国王 Harald Bluetooth 的名字，寓意"统一通信协议"
- **Bug**：1947 年真的有飞蛾卡在哈佛计算机里

## 5. 核心感悟

> **历史是万物的注脚。**

几乎任何领域深挖下去，都会回到历史：
- 键盘 QWERTY 布局 → 打字机时代防止卡键
- 美国用英制单位 → 独立战争后的身份认同
- Tab vs 空格之争 → 早期编辑器的技术限制

**技术圈看似高冷，背后都是普通人在赶 deadline、蹭热点、开玩笑、随便起名。**

代码是人写的，命名是人起的，历史是人创造的——所以总带着人的随意、幽默和无奈。
