## Phase 1：固定映射表（7档）

| 语言标签 | 数值 | 区间 |
|----------|------|------|
| +VH (Very High) | +0.9 | [0.8, 1.0] |
| +H (High) | +0.7 | [0.6, 0.8] |
| +M (Medium) | +0.5 | [0.4, 0.6] |
| +L (Low) | +0.3 | [0.2, 0.4] |
| 0 (No effect) | 0.0 | [0.0, 0.0] |
| -L | -0.3 | [-0.4, -0.2] |
| -M | -0.5 | [-0.6, -0.4] |
| -H | -0.7 | [-0.8, -0.6] |
| -VH | -0.9 | [-1.0, -0.8] |

**选择理由**：简单、一致性好、易解释。LLM直接输出数值校准差，不采用。
