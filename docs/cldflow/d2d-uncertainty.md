# D2D层 - 不确定区间计算

> 架构概览见：`docs/CLDFlow-architecture.md` D2D层
> 默认参数见：`docs/CLDFlow-defaults.md` D2D层

---

## 不确定性来源

| 来源 | 量化方式 |
|------|----------|
| Agent分歧 | 权重分布范围 |
| 语言标签粒度 | 映射表自带区间（如+H=[0.6,0.8]） |
| 证据质量 | 证据置信度 |
| 模型简化 | 保守估计区间 |

---

## Phase 1：极值区间

取所有Agent意见的最小值和最大值。

```
政策Agent: +H (0.7)
经济Agent: +M (0.5)
社会Agent: +H (0.7)

→ 区间: [0.5, 0.7]
→ 点估计: 0.6 (均值)
→ 不确定半径: 0.1
```

---

## Phase 2：贝叶斯可信区间

从极值区间升级为概率分布，用Agent分歧推断后验。
