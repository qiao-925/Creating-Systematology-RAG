| **检索层** | 向量化、索引、检索 | Embedding模型、向量索引、检索策略、重排序 |

| **应用层** | UI、部署、运维 | 用户界面、部署配置、生产优化 |
| **评估层** | 可观测性、评估 | Phoenix集成、RAGAS评估、性能监控 |
