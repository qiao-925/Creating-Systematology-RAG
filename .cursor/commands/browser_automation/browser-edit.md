# /browser-edit

## 1. Command Summary
- 目的：使用浏览器可视化编辑器进行 UI 调整，自动应用更改到代码
- 触发时机：需要调整 UI 样式、布局或交互时
- 关联规则：`browser_visual_editor_integration.mdc`
- 依赖工具：`@browser`（可视化编辑器）

## 2. 输入
- 必需：编辑目标描述（如"调整聊天输入框圆角为 20px"）
- 可选：目标元素选择器、具体属性值

## 3. 执行步骤
1. 确保应用已启动（如未启动，先执行 `/browser-start`）
2. 使用 `@browser` 打开应用页面
3. 根据描述定位目标元素
4. 使用可视化编辑器进行调整：
   - 位置和布局（flex、grid、定位）
   - 尺寸（宽度、高度、内边距、外边距）
   - 颜色（背景色、文字色、边框色）
   - 外观（阴影、透明度、圆角）
5. 实时预览效果
6. 用户确认后，点击"应用"按钮
7. Agent 自动更新对应代码文件（通常是 `frontend/main.py` 的 CSS 部分）
8. 验证更改是否正确应用（热重载后检查）

## 4. 输出
- 编辑操作摘要
- 更新的代码文件路径
- 代码变更摘要
- 验证结果（成功/失败）

## 5. 示例
```
用户：/browser-edit 调整聊天输入框圆角为 20px，边框颜色改为 #6366f1

执行：
1. 打开浏览器
2. 选中聊天输入框
3. 调整圆角：20px
4. 调整边框颜色：#6366f1
5. 应用更改
6. 更新 frontend/main.py 中的 CSS
7. 验证效果
```
